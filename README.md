# Weatherly v2 - Linux Fork

The official Linux port of [Weatherly v2](https://github.com/Wjanek13/Weatherly-v2).

## Requirements
- Python 3.x
- pyowm
- Pillow
- requests

## Installation
```bash
pip install pyowm pillow requests
```

## Usage
```bash
python script.py
```
